* Introduction

~pybrat~ is a reader/parser for reading/parsing data annotated by [[https://brat.nlplab.org/index.html][brat]].

* Installation

** From pip

#+begin_src sh
pip install pybrat
#+end_src

** From source

#+begin_src sh
pip install git+https://github.com/Yevgnen/pybrat
#+end_src
